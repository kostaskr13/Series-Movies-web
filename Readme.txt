Movie/Series App
This is a simple React application built with TypeScript that allows you to search and sort movies or series using the OMDb API.

Features:
Search movies or series based on the selection of MOVIES or SERIES.
Sort items by year, alphabetically, or rating.
Clicking on an item redirects to a details page with information about the movie or series.
Toggle between light and dark mode .

The project structure is set up as follows:

src: Contains the application source code.
components: Contains the React components for different parts of the application.
styles: Contains CSS/SCSS files for styling the components.
App.tsx: Sets up routing and renders the main components.

The project uses the following dependencies:

axios: For making HTTP requests to the OMDb API.
react-router-dom: For handling routing in the application.
